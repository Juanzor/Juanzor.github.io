Perfumall

Pagina web de compra y venta de perfumes desarrollada en equipo usando tecnologias como Javascript, Node, Express, MySql, Sequelize, Template Engines, Bootstrap, etc. Registro y login de usuarios comunes y administradores, los cuales podran cargar sus productos a la base de datos. Asi mismo, se realizó una api para consumirla en un dashboard con React. Trabajo realizado en un entorno con metodologías ágiles

ShortUrl

Acortador de URLS, en el que el usuario podrá registrarse, confirmar su cuenta con su mail, y loguearse. Tener guardadas sus propias urls acortadas y un perfil donde podra consultar sus datos y foto perfil. Para lograr esto todo esto, se empleó passport, formidable, jimp bcryptjs, nodemailer, express-validator, csrf, Template Engines, etc. Los usuarios y sus sesiones activas se almacenan en MongoDB.

Dashboard Perfumall

Métricas sobre el sitio Perfumal hecho con React, en el que muestra estadisticas tales como la cantidad de productos, marcas, usuarios, búsqueda de usuarios y productos, último usuario y producto registrados, cantidad de productos por marca, etc. Para mostrar estas analiticas, se realizó una serie de algoritmos sobre los datos consumidos por la api de Perfumall intentando hacer la menor posible de peticiones a la base de datos.
